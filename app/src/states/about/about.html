How it works:

AudibleHero will;
  1. load your library.
  2. find all authors you have purchased books from.
  3. find all books published by these authors.
  4. group books by series.
  5. group books by authors.
  6. repeat 2.- 5. with incognito mode.

The list of books is stored on your computer, so if you go to another computer you have to run update again.
List of ignored books are stored on your Google account, so it will be shared between computers.
It is nevertheless wise to run update every time you open AudibleHero looking for a new book, or else AudibleHero will not know about them.

Incognito

AudibleHero will run the update twice, once as your user and once as incognito.
Reason is that books you can see are regional based. If you do a search in incognito you will see all books.
It is possible to buy books cross regionality.
In order to determine if a book appeared only when in incognito the update is run twice.

Ignored books

AudibleHero let you ignore books. This feature let you "tag" books you don't want to show up on the front page.
The reason can be that you don't want to ever read the book, or any of the books by an author.
This feature was added because it was hard to see which books were really new books in series, and which were not.
While the list of books is stored on your computer, the list of ignored books are stored on your Google account.
When you run update on another computer the ignored books stays ignored.

FAQ

Will AudibleHero make changes to my Audible page and/or account?
No.

How often is the library updated?
The library is updated only manually via the update page.

AudibleHero doesn't find all books in my library.
Some books are removed from Audible. It is possible to open them with a direct link, but they will no longer show up in search results.

AudibleHero support authors with same name?
AudibleHero use the author names to find books. Connection to series will work fine, but AudibleHero can't distinguish between two different authors with the same name.
As of today there is no distinct identifier between authors on Audible beside their name.
